# semaforo-inteligente

- Nome do projeto: OkSemáfro - SP
- Nome do desenvolvedor: Kenneth Anderson Pedrosa Cavalcante
- Objetivo do projeto: Reduzir os acidentes de trânsito e organizar o tráfego de veículos no trânsito em São Paulo.
- Linguagem utilizada: C#
- Descrição resumida do funcionamento: O semáforo deve dividir as vias entre principal e secundária. Os veículos só poderão seguir adiante se o semáforo da sua via estiver verde e ficar atente quando estiver amarelo, caso contrário ele deve parar e esperar o tempo de ciclo do semáforo inteligente.
- Como executar o programa, quando houver código: Utilizando o compilador GCC, rodar o projeto dentro do Visual Studio Code para executar o programa.
